💡 TIPS & TRICKS

Build with BuildKit (faster):
   $ DOCKER_BUILDKIT=1 docker build -t android-tests .

Run with custom memory:
   $ docker run -m 4g --rm android-tests

Access container shell:
   $ docker-compose run android-tests /bin/bash

Check emulator status:
   $ docker run -it --rm android-tests \
     /bin/bash -c "adb devices; adb shell getprop ro.build.version.release"
